materials
hook size 2.5mm
two 6mm safety eyes
yarn: yarn art jeans (50gram)
stuffing (polyfill, etc) scissors,
abbrieviation
mr magic ring
ch chain
sc single crochet
inc single crochet increase
dec single crochet decrease
st stitch/ stitches
slst slip stitch
dc/dc double crochet
arm*2
start with red yarn,
rnd 1: start 7sc in a magic ring (7)
rnd 2: 7sc (7)
change to white yarn.
rnd 3: 7sc (7)
change to red yarn.
rnd 4: 7sc (7)
change to main color yarn.
rnd 5-8: 7sc (7)
fasten off, leaving a long tail for sewing.
leg*2
start with yarn,
rnd 1: start 6sc in a magic ring (6)
rnd 2: (2sc,inc)*2 (8)
change to white yarn.
rnd 3: 8sc (8)
change to red yarn.
rnd 4: 8sc (8)
change to main color yarn.
rnd 5-6: 8sc (8)
do not cut the yarn, work s chains, we will join both legs to croc-
body
rnd 7: 8sc+ ssc+ 8sc+ 5sc= 26sc (26)
rnd 8: 2sc, (inc,ssc)*4 (30)
rnd 9-14: 30sc (30)
rnd 15: 4sc,dec,(8sc,dec)*2, 4sc (27)
rnd 16: 27sc (27)
rnd 17: (7sc,dec)*3 (24)
fasten off, leaving a long tail for sewing.
head
rnd 1: start 6sc in a magic ring (6)
rnd 2: 6inc (12)
rnd 3: (1sc, inc)*6 (18)
rnd 4: (2sc, inc)*6 (24)
rnd 5: (3sc, inc)*6 (30)
rnd 6: (4sc, inc)*6 (36)
rnd 7: (5sc, inc)*6 (42)
rnd 8: (6sc, inc)*6 (48)
rnd 9-17: 48sc (48)
rnd 18: 3sc, dec(6sc,dec)*5, 3sc (42)
rnd 19: (4sc, dec)*6 (36)
rnd 20: 2sc, dec(4sc, dec)*5, 2sc (30)
rnd 21: (3sc, dec)*6 (24)
fasten off, leaving a long tail for sewing.
insert the eyes between rounds 13&14, 8sc apart.
glaze
rnd 1: start 6sc in a magic ring (6)
rnd 2: 6inc (12)
rnd 3: (1sc, inc)*6 (18)
rnd 4: (2sc, inc)*6 (24)
rnd 5: (3sc, inc)*6 (30)
rnd 6: (4sc, inc)*6 (36)
rnd 7: (5sc, inc)*6 (42)
rnd 8: (6sc, inc)*6 (48)
rnd 9-12: 48sc (48)
rnd 13: 1sc,(skip 2st,5dc into 1st, skip 2st, 1sl,1sc)*7, slst.
fasten off, leaving a long tail for sewing.
